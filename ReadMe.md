Web Application

STOCK WATCHLIST

FEATURES : 
 1. DISPLAYS THE LIST OF STOCK TICKERS AND STOCK NAME STOCK PRICES
 2. ABILITY TO ADD THE FAVORITE STOCKS TO MY FAVOURITE STOCK LIST
 3. ABILITY TO REMOVE THE FAVOURITE STOCKS TO UN FAVOURITE
